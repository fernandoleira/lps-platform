The Locator Pointer System (LPS) platform is a mapping pointer app for IoT devices.

<!-- GETTING STRATED -->
## Getting Started

```
git clone https://github.com/fernandoleira/lps-platform.git
cd lps-platform
pip install --upgrade pip
pip install -r requirements.txt
flask run
```
